# TeamChess

**Collaborative turn-based chess with Stockfish-powered move selection.**

## Structure

- `shared/` — shared types and enums
- `server/` — Socket.IO + Stockfish server (TS)
- `client/` — React + Vite front-end

## Quickstart (local)

```bash
# root
npm install
npm run dev        # starts server & client
```

## Build

```bash
npm run build      # builds all workspaces
```
